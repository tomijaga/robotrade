Running the program
In the terminal input the following in order:
- 'npm install'
- "npm run start'

"Node version has to Be v13 or higher"

Right now the app served as a Single Page Application
with the help of express and react-router-dom.

-   Search for A stock, click the desired one from the drop
    down menu, and you get redirected to veiw the chart of
